## 명시적 형변환
> * 오류를 막기위해 다른언어들이 지원하는 '암시적 형변환'은 지원하지않음

### 형변환 함수들.
```Kotlin
toByte()
toShort()
toInt()
toLong()
toFloat()
toDouble()
toChar()
//등등..
```

#### 예제코드
```Kotlin
fun main() {
    var a:Int = 54321
    var b:Long =a.toLong()
    println(b)
}
```

## 배열
> * Array 클래스를 통해 지원.

##### 배열선언법
```Kotlin
fun main() {
    /* 배열선언 */
    var a = arrayOf(1,2,3,4,5)
    println(a[1])
    
    /* 특정크기를 가지는 배열 선언 */
    var nullArr = arrayOfNulls<Int>(5)
    println(nullArr[0])
    
}
```
